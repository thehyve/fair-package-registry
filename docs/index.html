FAIR Package Registry PoC

This project is a Proof of Concept for the FAIR Package Registry, developed by the Dutch Health Care Institute (Zorginstituut Nederland) and The Hyve. It provides ontologies, taxonomies, and example data for FAIR Package Statements in healthcare.

  • GitHub Project Root — View the source code, scripts, and documentation on GitHub.
  • Ontology Documentation — HTML documentation for the FAIR Package Registry ontology.
  • Taxonomy Documentation — HTML documentation for the complete taxonomy.

See the README for more details.
